Projects

Here are some of my projects that I have been working on lately. They exist as repositories on Github, so check them out to learn more.

Water Drop Controller

ESP32 Based bluetooth connected solenoid controller for use with Water Droplet Photography. This project includes all the source files used to produce the board, as well as a 3D printed enclosure.

WiFi Packet Visualizer

This small board uses an ESP8266 to sniff IP packets in the air and display how many there are per second on an 8-LED Display. Feel free to use the design for any WiFi based 8-bit binary display.
